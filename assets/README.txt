## Readme
##

Reminders es una aplicación para la gestión de tareas, con soporte para varios listados y
clasficiación de los elementos por fechas, priodridades. etc.


El objetivo de esta aplicacion es simplemente aprender a programar para la plataforma android,
utilizando teconologias como java, sqllite, las APIs de Androiod y por los entornos de desarrollo
que ofrece androidSDK y IDEs como Eclipse.


La aplicacion se divide en dos partes:

    Gestión de listas que te permite añadir, eliminar, renombrar, ordenar y limpiar las diferentes listas que
    puedas ir necesitando.

    Gestión de recordatorios que te permite añadir, eliminar, modificar los elementos de las listas,
    asignación prioridades, descripición y fechas y definicion de su estado acutal (Pendiente o Completado).
    
    La información con la que trabaja la aplicación es persistente y se almacena usando una base de datos SQLite. 
    Además de los datos, también cuenta con un sistema para gestionar las preferencias de la aplicación.
